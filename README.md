# Personal dotfiles

This repository simply contains my configurations and settings for personal
conveniance.

# Install

Requires [GNU stow](https://www.gnu.org/software/stow/).

```sh
# One at a time
stow bash zsh tmux ...

# Or all at once
for f in *(/); stow $f
```
